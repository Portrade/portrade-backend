# Portrade Backend

<br>

## 🔍Overview

### Portrade

Portfolio와 Trade의 합성어로 포트폴리오를 교환하는 웹 프로젝트입니다.

![main image 1](https://user-images.githubusercontent.com/52854045/143671142-d44c9482-a49b-423c-bdd9-6558ac6b531b.png)

### 개발 인원
5인(프론트엔드 3인, 백엔드 2인)

### 개발 기간
2021-09-12 ~

### 개발 목표
- 테스트 코드
- CI/CD 환경 구축
- RESTful API
- GitHub 활용

<br>

## 🛠️Tech Stack

### Backend
- Java11
- Gradle
- Spring Boot
- Spring Security
- Spring Data JPA
- Swagger

### Database
- MySQL
- H2

### DevOps
- AWS - EC2(Deploy server)
- AWS - LightSail(Build server)
- AWS - S3(File upload)
- Jenkins(CI/CD)
- Docker
